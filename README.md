## Getting started

**Follow these steps for use with _JSDelivr_:**

1. Go to your html/js file.
2. In this repo, go to [`src`](src)
3. For HTML enter this `<script>` tag:
   ```html
   <script src="https://jsdelivr">
   ```
4. For JS enter this
   ```js
   const utils = require([location of files]);
   ```
